# Tushar Sharma — Portfolio

A single-page interactive portfolio website built with HTML, CSS, and vanilla JavaScript (animated network background, scroll reveals, hover effects).

## 🚀 Deploy on GitHub Pages (free, live in minutes)

### Option A: New repository
1. Go to [github.com/new](https://github.com/new) and create a repo named **`portfolio`** (or anything you like).
2. Upload `index.html` to the repo (drag & drop via "Add file → Upload files").
3. Go to **Settings → Pages**.
4. Under "Build and deployment", set **Source** to `Deploy from a branch`, branch `main`, folder `/ (root)`. Save.
5. Wait 1–2 minutes — your site will be live at:
   ```
   https://<your-username>.github.io/portfolio/
   ```

### Option B: Profile site (root domain)
1. Create a repo named exactly **`<your-username>.github.io`** (e.g. `TusharSharmaji.github.io`).
2. Upload `index.html` to the repo root.
3. Pages activates automatically — live at:
   ```
   https://<your-username>.github.io/
   ```

## ✏️ Editing later
All content (text, links, skills) is plain HTML/CSS in `index.html` — open it in any editor, edit, and re-upload (or push via git) to update the live site.

## 🔗 Update your links
Your resume's GitHub and LinkedIn links are already wired into the Contact section. Update `mailto:`, `tel:`, GitHub, and LinkedIn URLs in `index.html` if they ever change.
